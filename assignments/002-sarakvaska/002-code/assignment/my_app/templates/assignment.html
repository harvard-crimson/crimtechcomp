Nick and Richard's Excellent Adventure

Instead of writing comp presentations themselves, Nick and Richard decided to travel through time in order to assemble history's finest computer scientists.

Unfortunately, they travelled back a bit too far... to the beginnings of the internet! The time travel police won't let them go back to the present age unless they build a website that's excellent.

What a totally good thing is, is that you have the power of modern technology and can make an awesome website to bring them back (unless you are using an outdated browser like IE 9, in which case they will be stuck in the past forever). Your tasks:

  • You should have a radical web inspector or a set of developer tools. Figure out how to open these, and check out the totally cool DOM elements. Maybe your first task is hiding somewhere...
  • Do you see that file `excellent.css`? Using inspect element, can you see how it gets linked to this page? (there's some Django-specific pieces to this if you look at the actual source code #spotthedifference) This link might help! Totally bodacious!
  • Like, after you've done that, find the id that marks your second task. It might contain some totally identical text. Nick or Richard might've made a major mistake. Fix it by using the right tag!
  • Read up about some absolutely awesome, totally Timothy people. For your third task, like, pick any computer scientist to write a sentence about. You should put this in a new paragraph element in the third task's div. Add the bodacious class to this paragraph element.
  • The internet police still won't let Nick and Richard back to the present because their website still looks totally terrible. In the most `excellent.css` file, make sure the margins of the website are 50 pixels on each side by changing the style rules for the wrapper.
  • Change the font to something way better.
  • The next task is to totally make this website more modern by adding some of the characteristic modern elements of a webpage.
  • Like, if we could have a nav bar and a footer, we'd be well on our way to at least 2005
  • And, like, if you could be so excellent as to add a logo to that navbar we'd be lookin' bodacious (make sure the logo has the id "logo")
  • Finally, play around and add 3 features you think make websites cool (I added a background image, I made a carousel, I added a form, etc., etc.) And list out those features in three fancy, bodacious bullet points below!
This is Margaret Hamilton, who wrote the code that took us to the moon.

Hamilton made up the term "software engineering" during the Apollo space mission day. When Hamilton started using the term "software engineering", software engineering was not taken seriously compared to other engineering, nor was it regarded as a science.
